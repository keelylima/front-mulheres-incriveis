O projeto

A ideia do projeto Subarashii Women é listar mulheres de sucesso em certas áreas do mercado e dar mais visibilidade a elas. A base do projeto é uma API que possui os métodos GET, POST, UPDATE e DELETE feita em Nodejs e consumindo dados do MongoDB.

Atenção, é apenas um projeto didático para o curso de Bootcamp Back-end da {reprograma}. API só ficará disponível com método GET para consulta de terceiros.
